Gemini CLI Slash Commands

Gemini CLI에서 사용 가능한 모든 명령어를 정리한 가이드입니다.

v0.42.0 기반

1. 세션 및 일반 제어

/help

CLI 사용법과 전체 명령어 목록을 표시합니다. (또는 /?)

/about

현재 CLI 버전 및 시스템 정보를 출력합니다.

/clear

화면과 시각적인 세션 기록을 지웁니다. (Ctrl+L)

/quit

CLI를 종료합니다. --delete 플래그로 기록 삭제 가능.

/stats

세션, 모델, 도구 사용 통계를 확인합니다.

Sub: session, model, tools

/copy

마지막으로 출력된 내용을 클립보드에 복사합니다.

2. 대화 및 세션 관리

/resume

세션 브라우저를 열거나 체크포인트를 관리합니다.

Sub: save, list, resume, share

/rewind

대화 기록을 탐색하며 이전 상태로 되돌립니다. (Esc 2번)

/restore

도구 실행 전으로 파일을 복구합니다.

/compress

대화 내용을 요약하여 컨텍스트 사용량을 줄입니다.

3. 설정 및 환경 구성

/settings

인터랙티브 설정 편집기를 엽니다.

/theme

CLI 테마를 변경하는 대화 상자를 표시합니다.

/model

사용할 모델을 설정하거나 관리합니다.

Sub: manage, set <model>

/directory

작업 디렉토리를 추가하거나 조회합니다.

/auth

인증 방법을 변경합니다.

/vim

Vim 입력 모드를 토글합니다.

4. 확장 및 고급 기능

/extensions

확장 기능을 설치, 활성화, 관리합니다.

/mcp

MCP 서버와 도구를 관리합니다.

/agents

서브에이전트 목록을 확인하고 설정합니다.

/skills

에이전트 스킬을 관리합니다.

/commands

사용자 정의 슬래시 명령어를 관리합니다.

5. 프로젝트 지원

/init

프로젝트 분석 후 GEMINI.md 파일을 생성합니다.

/memory

GEMINI.md 기반의 지시 기억을 관리합니다.

/plan

현재 작업 계획을 보거나 클립보드에 복사합니다.

/bug

버그 리포트를 제출합니다.

/docs

공식 문서를 브라우저에서 엽니다.
